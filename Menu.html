CSS


es el lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML (incluyendo varios lenguajes basados en XML como SVG, MathML o XHTML). CSS describe cómo debe ser renderizado el elemento estructurado en la pantalla, en papel, en el habla o en otros medios.

Atributos

Nombre del Atributo Descripción Valores Posibles Ejemplo de Uso Imagen Representativa Fuente de Información
font-size La propiedad font-size especifica la dimensión de la letra. Este tamaño puede, a su vez, alterar el aspecto de alguna otra cosa, ya que se usa para calcular la longitud de las unidades em y ex. px, %, em, rem font-size: 16px;
Fuentes
border-radius Redondea las esquinas de un elemento. px, % border-radius: 10px;
Fuentes
display Define cómo se debe mostrar un elemento en la página. block, inline, inline-block, flex, grid, none display: flex;
Fuentes
flexbox Utiliza Flexbox para diseñar elementos en un contenedor flexible. flex-direction, justify-content, align-items display: flex; justify-content: center;
Fuentes
position Especifica el método de posicionamiento de un elemento. static, relative, absolute, fixed, sticky position: absolute; top: 50px;
Fuentes
box-shadow Aplica una sombra a un elemento. Valores de desplazamiento, desenfoque, color box-shadow: 5px 5px 15px rgba(0,0,0,0.5);
Fuentes
overflow Aplica una sombra a un elemento. Valores de desplazamiento, desenfoque, color box-shadow: 5px 5px 15px rgba(0,0,0,0.5);
Fuentes
z-index Define el orden en el eje z de los elementos (superposición). Números enteros z-index: 10;
Fuentes
opacity Establece el nivel de transparencia de un elemento. Valores entre 0 (transparente) y 1 (opaco) opacity: 0.5;
Fuentes
margin y padding El margin define el espacio exterior mientras que el padding el espacio interior de un elemento. Valores en px, %, auto margin: 10px; padding: 20px;
Fuentes
text-align Controla la alineación del texto dentro de un elemento
  • Valores absolutos: xx-small, x-small, small, medium, large, x-large, xx-large
  • Valores relativos: larger, smaller
  • Unidades de longitud
  • Porcentajes (en relación al elemento padre)
  • REM (Root Em), una unidad relativa al elemento raíz HTML
Fuentes
